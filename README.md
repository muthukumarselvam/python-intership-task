# python-intership-task
 Python Internship Tasks Project

📌 Project Overview

This repository contains Python programs completed as part of a Python Internship.
The tasks are divided into three levels: Easy, Medium, and Hard.

The objective of this project is to build a strong foundation in Python programming, problem-solving, and basic application development.

✅ Tasks Completed

🔹 Easy Level Tasks (3)

- Simple Calculator
- Number Guessing Game
- Multiplication Table Generator

🔹 Medium Level Tasks (2)

- Password Generator
- Contact Book Application

🔹 Hard Level Task (1)

- To-Do List Application
